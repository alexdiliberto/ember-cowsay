## Installation

* `git clone <repository-url>` this repository
* `cd ember-cowsay`
* `npm install`
* `bower install`

## Running Tests

* `npm test`

## Building

* `ember build`

## Release

```sh
# v0.7.0
ember release
# v0.7.1
ember release --minor
# v0.8.0
ember release --major
# v1.0.0
```

For more information on using ember-cli, visit [http://ember-cli.com/](http://ember-cli.com/).
